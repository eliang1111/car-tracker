# 🛠️ คู่มือการตั้งค่าระบบ Car Tracker (LINE OA + Google Sheets)

คู่มือนี้จะอธิบายขั้นตอนการติดตั้งระบบบันทึกเติมน้ำมันและซ่อมบำรุงรถยนต์ของคุณทีละขั้นตอนอย่างละเอียดครับ

---

## 📊 ขั้นตอนที่ 1: ตั้งค่า Google Sheets & Apps Script (ฐานข้อมูลหลังบ้าน)

1. เปิดเบราว์เซอร์แล้วไปที่ [Google Sheets](https://sheets.google.com) จากนั้นสร้างสเปรดชีตเปล่าขึ้นมาใหม่ 1 ไฟล์
2. ตั้งชื่อสเปรดชีต เช่น `Car-Tracker-Database`
3. ที่แถบเมนูด้านบน ไปที่ **Extensions (ส่วนขยาย)** > **Apps Script**
4. ลบโค้ดเดิมทั้งหมดในไฟล์ `Code.gs` ทิ้ง
5. เปิดไฟล์ [Code.gs](file:///d:/APP/Car-Tracker/Code.gs) ในโปรเจกต์นี้ คัดลอกโค้ดทั้งหมดไปวางในช่องบรรณาธิการของ Google Apps Script
6. กดปุ่ม 💾 **Save** (รูปแผ่นดิสก์)
7. กดปุ่ม **Deploy (การใช้งานได้จริง)** ที่มุมบนขวา > **New deployment (การใช้งานได้จริงใหม่)**
8. ตั้งค่าดังนี้:
   * **Select type**: คลิกรูปฟันเฟือง เลือก **Web app (เว็บแอป)**
   * **Description**: พิมพ์คำอธิบาย (เช่น `V1.0.0`)
   * **Execute as**: เลือก **Me (อีเมลของคุณ)**
   * **Who has access**: เลือก **Anyone (ทุกคน)** *(สำคัญมาก เพื่อให้ไลน์เข้าถึงได้)*
9. กดปุ่ม **Deploy**
10. ระบบอาจจะขึ้นหน้าต่างให้กดยอมรับสิทธิ์ (Authorize access) ให้กดเลือกบัญชี Google ของคุณ -> กด **Advanced** -> กด **Go to ... (unsafe)** -> กด **Allow** เพื่ออนุญาตสิทธิ์
11. เมื่อเสร็จสิ้น คุณจะได้ **Web App URL** (ลิงก์ยาวๆ ที่ลงท้ายด้วย `/exec`) **ให้คัดลอกลิงก์นี้เก็บไว้** 📋

---

## 🌐 ขั้นตอนที่ 2: ตั้งค่า GitHub Pages (ฝากไฟล์หน้าเว็บฟรี)

1. สมัครใช้งานหรือเข้าสู่ระบบที่ [GitHub](https://github.com)
2. สร้าง Repository ใหม่ ตั้งชื่อเป็น `Car-Tracker` (สามารถตั้งเป็น Public หรือ Private ก็ได้)
3. อัปโหลดโฟลเดอร์โปรเจกต์นี้ขึ้น Repository ของคุณ โดยก่อนอัปโหลด ให้ทำการแก้ไขไฟล์ดังนี้:
   * เปิดไฟล์ [js/api.js](file:///d:/APP/Car-Tracker/js/api.js) บนคอมพิวเตอร์ของคุณ
   * นำ **Web App URL** ที่ก๊อปปี้มาจาก *ขั้นตอนที่ 1 ข้อ 11* ไปวางทับคำว่า `'YOUR_APPS_SCRIPT_WEB_APP_URL'` ในบรรทัดที่ 9
   * บันทึกไฟล์
4. เมื่ออัปโหลดไฟล์ทั้งหมดขึ้น GitHub เรียบร้อยแล้ว (`fuel.html`, `maintenance.html`, `history.html`, โฟลเดอร์ `css`, โฟลเดอร์ `js`)
5. ที่หน้า Repository ใน GitHub ไปที่เมนู **Settings (การตั้งค่า)** > ด้านซ้ายเลือก **Pages**
6. ในหมวด **Build and deployment**:
   * **Source**: เลือก **Deploy from a branch**
   * **Branch**: เลือก **main** (หรือ master) และเลือกโฟลเดอร์ `/ (root)`
   * กด **Save**
7. รอประมาณ 1-2 นาที ให้รีเฟรชหน้าเว็บ คุณจะได้ลิงก์หน้าเว็บ GitHub Pages ของคุณ เช่น:
   * `https://<ชื่อผู้ใช้>.github.io/Car-Tracker/fuel.html`
   * `https://<ชื่อผู้ใช้>.github.io/Car-Tracker/maintenance.html`
   * `https://<ชื่อผู้ใช้>.github.io/Car-Tracker/history.html`
   * **จดจำหรือก๊อปปี้ลิงก์ทั้ง 3 หน้านี้ไว้**

## 💬 ขั้นตอนที่ 3: ตั้งค่า LINE Developers & LIFF Apps

1. ไปที่เว็บไซต์ [LINE Developers Console](https://developers.line.biz/console/) และเข้าสู่ระบบด้วยบัญชี LINE ของคุณ
2. สร้าง **Provider** ใหม่ (หรือเลือกอันเดิมที่มีอยู่)
3. กดสร้าง **Channel** ใหม่ โดยเลือกประเภทเป็น **LINE Login**
4. ตั้งชื่อ Channel และอัปโหลดรูปภาพตามต้องการ จากนั้นกดสร้าง
5. คลิกเข้าไปที่ Channel ที่เพิ่งสร้าง แล้วเลือกแท็บ **LIFF** ด้านบน
6. กดปุ่ม **Add** เพื่อสร้าง LIFF App ทั้งหมด **3 ตัว** โดยตั้งค่าทีละตัวดังนี้:

### 📱 LIFF ตัวที่ 1: เติมน้ำมัน
* **LIFF app name**: `เติมน้ำมัน`
* **Size**: เลือก **Tall** (หรือ Full ก็ได้)
* **Endpoint URL**: ลิงก์จาก GitHub Pages ของคุณ: `https://<ชื่อผู้ใช้>.github.io/Car-Tracker/fuel.html`
* **Scopes**: ติ๊กถูกที่ `profile` และ `openid`
* กด **Save**

### 📱 LIFF ตัวที่ 2: บันทึกซ่อมบำรุง
* **LIFF app name**: `บันทึกซ่อมบำรุง`
* **Size**: เลือก **Full**
* **Endpoint URL**: `https://<ชื่อผู้ใช้>.github.io/Car-Tracker/maintenance.html`
* **Scopes**: ติ๊กถูกที่ `profile` และ `openid`
* กด **Save**

### 📱 LIFF ตัวที่ 3: ประวัติและค้นหา
* **LIFF app name**: `ดูประวัติ/ค้นหา`
* **Size**: เลือก **Full**
* **Endpoint URL**: `https://<ชื่อผู้ใช้>.github.io/Car-Tracker/history.html`
* **Scopes**: ติ๊กถูกที่ `profile` และ `openid`
* กด **Save**

## 🔗 ขั้นตอนที่ 4: นำรหัส LIFF IDs มาใส่ในโค้ดและอัปโหลดอีกครั้ง

1. หลังจากสร้าง LIFF ทั้ง 3 ตัวในไลน์เสร็จแล้ว คุณจะได้ **LIFF ID** (เช่น `2001234567-aBcDeFgH`) ของแต่ละตัวมา
2. เปิดไฟล์ [js/app.js](file:///d:/APP/Car-Tracker/js/app.js) บนคอมพิวเตอร์ของคุณ
3. นำรหัส LIFF ID ของแต่ละหน้ามาใส่แทนที่ตัวแปรในบรรทัดที่ 10-14:
   ```javascript
   const LIFF_IDS = {
     fuel: 'ก๊อปปี้รหัส LIFF ID เติมน้ำมันมาใส่ที่นี่',
     maintenance: 'ก๊อปปี้รหัส LIFF ID ซ่อมบำรุงมาใส่ที่นี่',
     history: 'ก๊อปปี้รหัส LIFF ID ประวัติมาใส่ที่นี่'
   };
   ```
4. บันทึกไฟล์ และอัปโหลดไฟล์ `js/app.js` ที่อัปเดตนี้ขึ้น GitHub Repository ตัวเดิมของคุณอีกครั้งเพื่อให้หน้าเว็บทำงานร่วมกับไลน์ได้อย่างสมบูรณ์

## 🎨 ขั้นตอนที่ 5: ตั้งค่า Rich Menu บน LINE Official Account

1. ไปที่เว็บไซต์ [LINE Official Account Manager](https://manager.line.me/) แล้วเลือกบัญชี LINE OA ของคุณ
2. ที่เมนูด้านซ้าย ไปที่ **ริชเมนู (Rich Menus)** > กดปุ่ม **สร้างใหม่ (Create new)**
3. ตั้งค่ารายละเอียดริชเมนูตามต้องการ
4. ในส่วน **การตั้งค่าเทมเพลต (Template settings)**:
   * เลือกเทมเพลตแบบ **3 ช่อง แถวเดียวกว้างเท่ากัน** (แถวล่างสุดซ้ายไปขวา)
5. ทำการอัปโหลดรูปภาพริชเมนูที่คุณเตรียมไว้ (รูปปั๊มน้ำมัน คลินิกรถยนต์ และแว่นขยายค้นหา)
6. ในส่วน **แอคชัน (Actions)** กำหนดการทำงานของทั้ง 3 ปุ่มดังนี้:
   * **ปุ่ม A (ซ้ายสุด - เติมน้ำมัน)**: 
     * **ประเภท (Type)**: เลือก **ลิงก์ (Link)**
     * **ลิงก์ URL**: ใส่ **LIFF URL** ของระบบเติมน้ำมัน (เช่น `https://liff.line.me/2001234567-xxxxxx1`)
     * **ป้ายแอคชัน (Action label)**: ใส่คำว่า `เติมน้ำมัน`
   * **ปุ่ม B (กลาง - บันทึกซ่อมบำรุง)**:
     * **ประเภท**: เลือก **ลิงก์**
     * **ลิงก์ URL**: ใส่ **LIFF URL** ของบันทึกซ่อมบำรุง (เช่น `https://liff.line.me/2001234567-xxxxxx2`)
     * **ป้ายแอคชัน**: ใส่คำว่า `บันทึกซ่อมบำรุง`
   * **ปุ่ม C (ขวาสุด - ดูประวัติ/ค้นหา)**:
     * **ประเภท**: เลือก **ลิงก์**
     * **ลิงก์ URL**: ใส่ **LIFF URL** ของดูประวัติ/ค้นหา (เช่น `https://liff.line.me/2001234567-xxxxxx3`)
     * **ป้ายแอคชัน**: ใส่คำว่า `ดูประวัติ`
7. กด **บันทึก (Save)** เพื่อเปิดใช้งานริชเมนู
